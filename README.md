# keyword-extractor-korean
Extract keyword from plain-text.

## Install

```js
npm install keyword-extractor-korean
```

## Test

```js
npm test
```

## Usage

```js
var keyword = require("../index.js");

keyword.extract('천자만홍이 없는 있는 과실이 황금시대를 황금시대다.');
// {'천자만홍': 1, '과실': 1, '황금시대': 2}
```
